Here’s some projects I worked on recently.

COVID-19 AI Image Generator

The COVID-19 Image Generation project develops a Convolutional Neural Network (CNN) to analyze medical scans and determine COVID or non-COVID status. The model is trained with labeled datasets for classification. A Generative Adversarial Network (GAN) generates synthetic COVID and non-COVID images to augment the dataset, improving model accuracy. The project uses Python, PyTorch, Gradio, Django, Jupyter Notebook, and GitHub.

AIVi Research Group: Vitual Reality

The research project focuses on creating a Virtual Reality (VR) experience for the Agriculture Department, built using Unity and Blender. Its purpose is to provide an educational tool for farmers and students, allowing them to learn agricultural concepts in an interactive virtual environment. The VR experience enhances learning offering an immersive and practical approach to education in the field of agriculture.

Bivariate MLE Dash Appliaction

The Bivariate MLE Dash Application is a web-based tool that allows users to perform Maximum Likelihood Estimation (MLE) analysis on bivariate data. It provides an interactive interface where users can input data, estimate model parameters, and visualize the results. The app integrates data processing, MLE calculations, and real-time visualizations, making it easy for users to analyze and interpret statistical data.

Lexical Analyzer

This project is a lexical and syntax analyzer that reads source code, breaks it into tokens, and checks if it follows a specified BNF grammar. It identifies elements like keywords and operators, then validates the syntax, printing errors if any are found.

Payment System Application

This Java program simulates a payment system for employees, including different types of staff: Lecturers, Assistant Professors, and Professors. The program allows users to input employee information, such as personal details and salary, and calculates their payments based on the type of employee. It also handles special cases, such as bonuses for professors and summer payments for assistant professors.

Airline Reservation System

A ticket system for four airplanes, allowing customers to purchase and return tickets. It handles the airplane details, including flight times and available seats. If tickets are unavailable, it throws an exception. Customers can choose to buy or return tickets, with seat numbers being managed accordingly. The system tracks sold and returned tickets, ensuring accurate seat availability.
